## 🚀 Featured Projects

**[WHO Global Disease Outbreak Pipeline](https://github.com/yourusername/who-outbreak-pipeline)**
End-to-end AWS data pipeline processing 3,500+ disease outbreak records across 90+ diseases and 236 countries. Built with Glue (PySpark), Redshift, and Power BI, with automated scheduling and CloudWatch monitoring.
`AWS Glue` `Redshift` `PySpark` `Power BI`

**[Azure Spotify Data Engineering Pipeline](https://github.com/yourusername/azure-spotify-project)**
Cloud-native incremental data pipeline using Azure Data Factory and Databricks with medallion architecture and SCD Type 2 dimension tracking.
`Azure Data Factory` `Databricks` `PySpark` `Delta Live Tables`

**[Stock Price Analytics Pipeline](https://github.com/yourusername/stock-price-analytics-pipeline)**
Medallion architecture (Bronze/Silver/Gold) pipeline with dimensional modeling, loaded into Snowflake for BI reporting.
`Databricks` `Snowflake` `Power BI`

**[Airbnb NYC Analytics Dashboard](https://github.com/yourusername/airbnb-nyc-analytics)**
Interactive analytics dashboard analyzing 41,697 NYC Airbnb listings with drill-down filters across neighborhoods and room types.
`Snowflake` `Power BI` `SQL`
